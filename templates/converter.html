Help & Features

This tool allows you to:

  1. Upload and decrypt save files from various games
  2. View and edit save data, WSRef data, or WSDir data
  3. Download the modified files back to your device
  4. Export the data as JSON for further analysis

Supported File Types:

  • Save Files (.txt): Game save data files that can be edited and modified
  • WSRef Files (.assets): Game reference data for exploring game assets
  • WSDir Files (.txt): Directory configuration files for game settings

How to Use:

  1. Select the appropriate tab for your file type
  2. Click "Browse Files" to select your file
  3. Click "Load" to process and display the data
  4. Edit the data in the JSON editor if needed
  5. Save changes or export as JSON

⚠️ WARNING: Always backup your original files before making changes. Modified save files may affect game compatibility.
